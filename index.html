기획서를 붙여넣으면
테스트 케이스가 만들어집니다

QA 업무 효율을 위해 만든 무료 도구입니다. 기획서(PRD) 텍스트를 입력하면 로그인·결제·검색 등 주요 기능을 인식해 표준 QA 양식의 테스트 케이스를 자동 생성합니다. 설치·로그인 없이 브라우저에서 바로 동작합니다.

즉시 동작 (정적 페이지) CSV · Markdown 내보내기 데이터는 브라우저 밖으로 나가지 않음

이렇게 동작합니다

예시를 고르고 변환 버튼만 누르면 테스트 케이스가 자동으로 만들어집니다.

QA PRD → TC
TC ID시나리오유형우선순위

1. 기획서 입력

기획서 파일을 여기로 끌어다 놓으세요

PPT(.pptx) · 워드(.docx) · 엑셀(.xlsx/.xls) · CSV · 텍스트 지원 — 또는 클릭해서 선택

예시:

키는 이 브라우저에서 Anthropic(api.anthropic.com)으로만 전송되며 별도 서버에 저장되지 않습니다. 키 발급: console.anthropic.com

단축키: Ctrl(⌘) + Enter

2. 생성 결과

총 0 정상 0 예외 0 경계 0

아직 생성된 테스트 케이스가 없습니다.
왼쪽에 기획서를 입력하고 테스트 케이스 변환 버튼을 눌러보세요.

TC ID 대분류 중분류 시나리오 사전조건 테스트 단계 기대결과 우선순위 유형

사용법 3단계

1

기획서 붙여넣기

기능 정의가 담긴 기획서 텍스트를 입력합니다. 제목과 항목으로 구조화할수록 결과가 정확합니다.

2

변환 실행

버튼을 누르면 주요 기능(로그인·결제·검색·업로드 등)을 인식해 정상/예외/경계 케이스를 자동 구성합니다.

3

검토 후 내보내기

표를 검토·보완한 뒤 CSV(엑셀) 또는 Markdown으로 내보내 TC 문서에 바로 활용합니다.

※ 자동 생성 결과는 초안입니다. 실제 적용 전 QA 담당자의 검토·보완을 권장합니다.

QA PRD → TC · 정적 페이지로 동작하는 오픈 도구 · 입력 데이터는 서버로 전송되지 않습니다.
